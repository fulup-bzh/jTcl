LOGIN page


This page login you on Rpc/Htttpd, login action will create a new private slave interpretor in httpd server, all rpc action will be done in this slave interpretor until you logout.
Note: login will create a cookie in your browser, this cookie will live 15mn, if you don't send any request in this space time, Rpc/Httpd will automaticaly log you out.
